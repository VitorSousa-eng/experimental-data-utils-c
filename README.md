# Objetivo:

  Este projeto tem como objetivo fornecer ferramentas práticas para automatizar o tratamento de dados laboratoriais cotidianos, comuns em disciplinas e atividades de laboratório experimental (como química e física experimental).
A motivação principal é reduzir cálculos repetitivos feitos manualmente, diminuindo erros humanos e aumentando a produtividade durante a análise de dados experimentais.

# Proposta do projeto:

  O projeto foi desenvolvido com foco em clareza, simplicidade e reutilização, seguindo três princípios centrais:

1️⃣ Ferramentas práticas para uso real

  As funções implementadas refletem cálculos recorrentes em ambientes laboratoriais, como médias, desvios e erros experimentais, permitindo que esses procedimentos sejam executados de forma rápida e confiável.

2️⃣ Código limpo e acessível para iniciantes

  As funções foram escritas de forma simples, direta e bem organizada, visando facilitar o entendimento de estudantes que estão tendo seus primeiros contatos com a linguagem C e com programação aplicada à engenharia.

3️⃣ Funções matemáticas reutilizáveis

Todas as funções:

- recebem valores numéricos como entrada;
- retornam valores numéricos (double);
- não realizam entrada ou saída diretamente.

  Isso permite que sejam facilmente reutilizadas dentro de outros programas, módulos ou funções, tornando o código flexível e extensível.

📌 Observação: 

Este é um projeto em estágio inicial, com foco educacional e formativo.
Novas funcionalidades, validações e melhorias poderão ser adicionadas conforme o avanço dos estudos e das aplicações práticas.
